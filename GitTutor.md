# This is my Git Tutorial

**Какие бывают команды**

* *git init* - команда, используемая для инициализации репозитория

* *git add* - команда, используемая для добавления изменений

* *git commit* - команда, которая окончательно сохраняет изменение + можно указать комментарий

* *git diff* - показывает, какие изменения были сделаны

* *git log* - показывает историю коммитов-сохранений

* *git checkout ...* - позволяет "перенестись" в определенный коммит-сохраниние

* *git checkout master* - возвращает нас к последнему коммиту в ветке

# Seminar 2

* Создание новой ветки и переходы между ветками

1. Для создания новой ветки используется команда *git branch xxx*, где ххх - название создаваемой ветки
2. Для перехода между ветками используется команда *git checkout xxx*, где ххх - название нужной ветки

* Слияние веток

1. Для осуществления слияния веток используется команда *git merge xxx*, где ххх - название ветки, сливаемой с текущей веткой

* Конфликты при слиянии, и их разрешение

1. Эта строка создана для демонстрации конфликта

1. Конфликты при слиянии возникают при наличии противоречащих коммитов в сливаемых ветках

* Добавление картинки

Перед тем как вставить картинку, нужно сохранить ее в папку с файлом, в котором мы работаем

2. Потом в квадратных скобках наименовать картинку
3. Потом в круглых скобках указать к ней путь

Если мы допустили ошибку в пояснении коммита, и его нужно переименовать, можно использовать команду *--amend*

# Семинар 3

Работа с удаленными репозиториями

Для того, чтобы начать работу с удаленными репозиторями, нужно зарегистрироваться на сайте [GitHub](https://github.com/), далее GH

## Отправка репозитория в GitHub

Чтобы создать удаленный репозиторий GH, нужно нажать на кнопку New и создать его) 

А затем нужно в VSCode сделать команды:

git remote add origin <ссылка на мой репозиторий на GH>
git branch -M main
git push -u origin main

И далее просто пользоваться командой git push, когда я вношу изменения в свой репозиторий. 

## Загрузка внешнего репозитория

Чтобы загрузить внешний репозиторий из GH, нужно сделать его Fork, скопировать ссылку на форкнутый репозиторий, нажать F1 в VSCode и в командной строке выбрать команду git clone.

Далее мы вставляем ссылку на удаленный форкнутый репозиторий, выбираем папку, куда его загрузим и можно работать
