# Lesson 1
 * *git init* - команда, инициализирующая репозиторий в текущей папке; 

 * *git status* - команда, отображающая текущее состояние репозитория;

 * *git add* **+/имя файла/**  добавить к отслеживанию файл;

 *  *git commit -m"название сохранения"* - команда, фиксирующая изменения в файле;

 *  *git log*  - журнал изменений;

 *  *git checkout +имя commit* - команда, для перехода на другое сохранение; 

 * *git checkout master* - команда, для перехода к актуальному состоянию;

 * *git diff* - команда, для проверки отличий в сохраненных версиях;

## Базовые термины первого семинара

* *git init* - команда, инициализирующая репозиторий в текущей папке;

* *git add* +*/имя файла/* - добавить к отслеживанию файл;

## Базовые термины второго семинара

* *git branch* + имя новой ветки   -создать новую ветку.

*Фаст форвард*  - технология слияния веток, когда одна "догоняет" другую. 
Т.е. вся информация подтягивается из второй.


**Что бы провести слияние веток необходимо находиться на ветке, куда будет добавлена новая ветка.**
* *git merge* + название ветки - команда для слияния веток.


     **Решения конфликтов слияния**.
     *  Accept Both Changes - оставить оба варианта. 
     *  Accept Current Change - оставить как было. 
     * Accept Incoming Change - принять входящие изменения. 





# Lesson 3

**GitHub**

Как скопировать репозиторий из GitHub.

* Кликнуть кнопку **Code** в GitHub.  Копировать ссылку на репозиторий.
* В терминале VSCode прописать git clone + *ссылка* ( Либо на экране 'начало работы', кликнуть "*клонировать репозиторий с GitHub*")
* В терминале прописать команду cd + имя папки(которую скачал).

Как отправить свой репозиторий в GitHub.

* Создать новый репозиторий в GitHub.
* Команда Git remote add origin + ссылка на репозиторий, в который нужно отправить.
* git branch -m master - команда указывающая как будет называться главная ветка
* git push -u origin master - отправить свой репозиторий.

Как использовать *"открытые"* репозитории или работа над одним проектом.

* В репозитории нажать кнопку Fork
* В терминале VSCode прописать git clone + *ссылка* ( Либо на экране 'начало работы', кликнуть "*клонировать репозиторий с GitHub*")
* В новой ветке создать файл README.md
