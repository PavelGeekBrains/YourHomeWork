# Шпаргалка по GIT`у #
## Основные команды
* *git init* - команда, инциализирующая локальный репозиторий
* *git version* - команда, отображающая текущую версию Git 
* *git status* - отображает состояние рабочего каталога и раздела проиндексированных файлов.
* *git add* - добавляет указанные файлы в индекс
* *git commit* - фиксирует добавленные в индекс изменения

## Команды веток
### Просмотр существующих веток
_git branch_ - просмотреть имеющиеся ветки + актуальную ветку 
### Создание новой ветки
_git branch **branch_name**_ - Создание новой ветки (*Branch_name* - Имя ветки).
### Перейти на другую ветку
_git checkout **branch_name**_ - Переход на другую ветку
### Слияние веток
_git merge **branch_name**_ - соединение двух разных веток
### Удаление ветки
_git branch -d **branch_name**_ - удалить выбранную ветку

## Работа с удаленным репозиторием

**Создать аккаунт на github.com. Сделать там удаленный репозиторий.**

**Кнопка "Fork" на github - копирование другого удаленного репозитория себе в аккаунт**

### Основные команды

**Если просто копируем репоизиторий на свой компьютер**

_git clone **ссылка на удаленный репозиторий**_ - Клонирает выбранный удаленный репозиторий на локальный

**Если соединяем удаленный репозиторий с локальным.**

1. _git remote add origin **ссылка на удаленный репозиторий**_ - соединение локального репозитория с удаленным

2. _git branch -M **branch_name**_ - Задать наименование главной ветки

3. _git push -u origin **branch_name**_ - отправка главной ветки удаленного репозитория главную ветку локального репозитория

### Другие команды

_git push_ - отправка информации из локального репозитория в удаленный

_git pull_ - отправка информации из удаленного репозитория в локальный
