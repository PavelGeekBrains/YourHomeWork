# Инструкция по работе с GIT

## Основные команды

* git init - создание репозитория в локальном каталоге

* git add - добавление файла для отслеживания изменений

* git commit - сохранение внесеннных в файл изменений

* git log - просмотр журнала действий с версиями файла

* git diff - контроль изменений в тексте

## Команды работы с ветками

* git branch - создание новых версий (веток)

* git checkout - переключение между разными версиями (ветками)

* git merge - слияние 2-х веток (версий) в одну


# Изучение системы GitHub

## Основные команды по работе в GitHub

* git clone - создание клона удаленного репозитория на локальной машине

* git pull - загрузка удаленного репозитория в локальный

* git push - отправка локальной версии репозитория вв внешний репозиторий

* pull request - запрос на слияние локального и внешнего репозитория
