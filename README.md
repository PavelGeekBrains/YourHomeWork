# Руководство по работе с Git

## Базовые термины первого семинара 

* *git init* - команда, инициализирующая репозиторий в текущей папке;

* *git add* - добавляет актуальную версию указанного файла к отслеживанию;

* *git status* - команда, отображающая текущее состояние репозитория; 

* *git log* - команда, отображающая commit в репозитории;

* *git commit -m* - команда для фиксации изменений с комментарием;

* *git checkout* - команда для перемещения между commit или ветками;

## Базовые термины второго семинара

* *git clear* - очистка терминала;

* *git branch* - команда, отображающая ветки;

* *git merge* - команда для слияние веток; 

* *git branch -b* - создание ветки и переход на неё;

* *git log --graph* - просмотр графа комитов;

* *git --help* - справка команд Git

* *git commit --amend* - коррекция сообщения к комиту;

## Базовые термины третьего семинара

* *git pull* - принять изменения из удаленного репозитория;

* *git push* - отправить в удаленный репозиторий;
