  # My git intraction

* *git init* команда иницилизирующая репозитурий в текучий момент

* *git add*-команда сохраняющая проєкт на текущей стадии сommits

* *Git commit*-команда фиксирующая текущую верссию проэкта

* *Git version*-команда показывающая текущую версию Git 

* *Git config*-кщманда позволяющая Git расспознать пользовотеля

* *Git status*-команда показывающая статус репозитория

* *Git log*-команда для просмотра истории commits

* *Git checkout*-команда позволяющая перемещатся мужду ветками

## Базавые термины первого семинара

* *Git init*-команда, иницилизирующая репозиторий

* *Git initial commit*-работа с Git начинается с первого Commit "Ininitial commit"
 
* *Git add*-добавляет актуальную версию указаного файла к отслеживанию

* *Extension md*-расширение файлов Markdown в основном встречается в версиях веб сайтов

* *Git diff*-команда, позволяющая сравнить файлы 

## Базавые термины второго семинара

* *Git brench*- команда отображающая список существующих веток

* *Git merge branch_name*-команда, команда остуществляющая слияние текущей ветки с branch_name
