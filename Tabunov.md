# Открываем терминал. 
Это можно сделать нажав сочетание клавишь Ctrl + Shift + `
(Последующие команды вводятся в нем);
# Сначала необходимо ввести данные о пользователе, сделать это можно командой:  
* *git config* --global user.name «Tabunov Alexander»;
# Вводим почтовый адрес пользователя, это можно сделать командой:
* *git config* --global user.email a.y.tabunov@gmail.com;
# После установки дополнения Git, проверим его работоспособноность командой:
git --version  (на экран терминала выводится сообщение о версии git);
# Создаем свой репозиторий, в котором будем работать. Это можно сделать командой:
* *git init*;
# Команда показывает состояния файлов в рабочем каталоге и индексе: какие файлы изменены, но не добавлены в индекс; какие ожидают коммита в индексе. 
* *git status*;
# Команда git add добавляет содержимое рабочего каталога в индекс (staging area) для последующего коммита.
* *git add* GitInstruction.md;
# Команда git commit берёт все данные, добавленные в индекс с помощью git add, и сохраняет их слепок во внутренней базе данных
* *git commit -m*"First commit";
# Команда git log используется для просмотра истории коммитов, начиная с самого свежего и уходя к истокам проекта. 
* *git log*;
# Выделение текста
Чтобы выделить текст курсивом, необходимо обрамить его звездочками (*). Например, *вот так*
или
Чтобы выделить текст полужирным, необходимо обрамить его двойными звездочками (**). Например, **вот так**
# Списки
Чтобы добавить ненумерованные списки, необходимо пункты выделить звездочкой (*). Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3
Чтобы добавить нумерованные списки, необходимо пункты просто пронумеровать. Например, вот так:
1. Первый пункт
2. Второй пункт
3. Третий пункт
#  Работа с изображением
Чтобы вставить изображение в текст, необходимо написать следующее:
![It's true](matrix.jpg) - but not at all
# Ссылки
Чтобы добавить ссылку в документ необходимо вставить ее в документ. Например, вот так:
https://gb.ru/lessons/245082/homework

# Подключаемся к удаленному репозиторию на github.com
* git branch -M main
* git remote add origin https://github.com/zaracis/ThirdSeminar.git
* git push -u origin main
* Отвечаем положительно на все всплывающие окна
