# My git tutorial -- Ivan

## Список изученных команд на 26.06.2022 (ненумерованный): 

* **"git init"** -- *команда инициализирует репозиторий в текущей папке*;

* **"git status"** -- *узнать текущий статус репозитория*;

* **"git add"** -- *добавление файла для отслеживания*;

* **"git commit"** -- *фиксация состояния файла (после внесения изменений в файл и сохранения его в качестве отслеживаемого)*;

* **"git diff"** -- *команда для проверки различий между текущим и зафикисированным состояниями файла*;

* **"git log"** -- *команда для просмотра истории коммитов (зафиксированных состояний файла)*;

* **"git checkout"** -- *команда для перехода между разными коммитами*;

* **"git branch"** -- *позволяет выводить список всех созданных веток*;

* **"git branch new_branch_name"** -- *создать новую ветку*;

* **"git branch -d branch_to_delete"** -- *удалить ветку*;

* **"git checkout branch_name"** -- *переход на другую ветку*;

* **"git log --graph"** -- *отображение графа (дерева) коммитов*;

* **"git merge branch_name"** -- *соединить ветки*;

* **"git clone"** -- *просим Git создать копию репозитория, который находится по ссылке <http...>, и можем указать название новой папки, в которую Git скопирует репозиторий <название папки>. Если его не указать, папка будет называться так же, как и сам репозиторий*;

* **"git push"** -- *используется для выгрузки содержимого локального репозитория в удаленный репозиторий. Она позволяет передать коммиты из локального репозитория в удаленный.*;

* **"git pull"** -- *Команда git pull используется для извлечения и загрузки содержимого из удаленного репозитория и немедленного обновления локального репозитория этим содержимым*;
